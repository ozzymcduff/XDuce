The Objective Caml distribution

The current version is 2.04, released 26/11/1999.

A newer beta version 2.99 has been released on 17/12/1999. Go to its distribution.

Sources:

  • Source distribution for the OCaml compilers and base libraries (.tar.gz format). Include the Unix, Win32 and Macintosh sources.
  • Source diffs from previous release (2.03).
  • Source for the OCamlTk graphical user interface (Unix and Win32).
  • Source RPM for the OCaml compilers + OCamlTk, ready to compile under Linux RedHat.
  • See the OCaml bazaar for contributed libraries and tools.

Precompiled binaries:

  • For Linux Intel x86 / RedHat 6.1 (RPM format).
  • For Linux Intel x86 / RedHat 6.0 (RPM format).
  • For Linux PowerPC 5 (RPM format).
  • For Linux Alpha / RedHat 6.0 (RPM format).
  • For Windows NT, 95, 98 (self-installing).
  • For MacOS 7, 8, and 9 (StuffIt archive).
  • For MacOS 7, 8, and 9 (BinHexed self-extracting archive).

User's manual and reference manual:

  • In PDF (free PDF viewers are available from Adobe).
  • In Postscript (compressed with gzip).
  • In DVI (compressed with gzip).
  • As a bundle of HTML files (in .tar.gz format).
  • As a bundle of HTML files (in zip format).
  • As a bundle of Emacs info files (in .tar.gz format).
  • As a plain text file.
  • The HTML version can be viewed online from the Caml Web site.

CVS access to the source tree:

Read-only CVS access to the whole Objective Caml and Caml Light sources (including working sources and all public releases) is available through our anonymous CVS server.
